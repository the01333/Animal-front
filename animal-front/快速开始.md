# i宠园前端 - 快速开始指南

## 📦 第一步：安装依赖

### Windows系统（推荐）
双击运行项目根目录下的 `install.bat` 文件

### 手动安装
```bash
npm install
```

## 🚀 第二步：启动开发服务器

```bash
npm run dev
```

启动成功后，浏览器会自动打开：`http://localhost:5173`

## 🌐 访问系统

### 前台页面
直接访问首页即可浏览前台内容：
- 首页：http://localhost:5173
- 宠物列表（瀑布流）：http://localhost:5173/#/pets

### 后台管理
访问管理后台需要登录：
- 管理后台入口：http://localhost:5173/#/admin/dashboard
- 需要使用管理员账号登录

## ❓ 常见问题

### Q: 安装依赖失败？
A: 尝试删除 `node_modules` 文件夹和 `package-lock.json`，然后重新运行 `npm install`

### Q: 端口被占用？
A: 修改 `vite.config.ts` 中的端口号

### Q: API请求失败？
A: 检查后端服务是否启动，API地址是否正确

### Q: 管理后台无法访问？
A: 确保使用管理员账号登录
